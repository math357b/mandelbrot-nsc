# Mandelbrot Set Generator

This project implements and compares different numerical methods for computing the Mandelbrot set, including:

- Naive Python implementation  
- NumPy vectorized implementation  
- Numba-accelerated implementation  
- Multiprocessing parallel implementation  
- Dask distributed implementation  
- GPU implementation for float32 and float64 

---

## Project Structure

- `figures/` -> Generated plots and images
- `lecture_exercises/` -> Exercise materials from lectures
- `lecture_py_files/` -> Shared helper modules used across scripts and notebooks
- `performance_notebooks/` -> Notebooks for performance benchmarking and analysis
